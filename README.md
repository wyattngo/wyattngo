## What I work on

- **ONFA Wallet V2** — a multi-product fintech monolith on CI3 / PHP / MySQL.
  Wallet, Exchange, P2P, NFT Mining, Staking, Futures, Savings, GameFi.
- **DrNick AI Agent** — async FastAPI / Python financial advisor with
  OpenAI GPT-4o (Claude fallback), Redis session memory, Postgres audit log,
  HMAC bridge to the PHP backend.
- **Workspace tooling** — multi-project Claude Code setup with custom hooks,
  skills, and an orchestrator that enforces internal conventions across
  every patch.

## How I work

- **Liquidity safety → user trust → token stability → growth.**
  That order. Every decision.
- **Small, safe patches.** STOP+WAIT after each one.
  Wallet code earns no benefit of the doubt.
- **Alignment over surgical hacks.** A parallel pattern today
  is tech debt tomorrow.
- **Read before you write.** Verify schema, then change it.
  Financial code does not forgive guesses.
